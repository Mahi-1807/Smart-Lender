# 📖 Overview

Traditional loan approval is often time-consuming and subject to manual evaluation. **Smart Lender** automates this process using Machine Learning models to provide fast, consistent, and accurate loan approval predictions.

# 🚀 Installation

### 📥 Clone Repository

```bash
git clone https://github.com/yourusername/Smart-Lender.git
```

### 📂 Move into Project

```bash
cd Smart-Lender
```

### 📦 Install Dependencies

```bash
pip install -r requirements.txt
```

### ▶️ Run Application

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:5000
```

# 💼 Use Cases

### 🚀 Fast-Track Loan Approval

A low-risk applicant with stable income and strong credit history receives instant approval prediction.

---

### ⚠️ High-Risk Applicant Detection

Applicants with poor credit history or unstable income are flagged for further verification.

---

### 📑 Bulk Loan Evaluation

Financial analysts can evaluate multiple loan applications quickly while maintaining prediction accuracy.
